# java-assessment
A simple application for answering a Java Interview Assignment.

## Tech Stack
- Java 21 (JDK 21 required)
- Spring Boot 4
- Maven
- MySQL 8
- Spring Data JPA (Hibernate)
- Flyway (database migrations)
- JUnit 5 (tests)

## Requirements
- JDK 21 installed and configured
- MySQL 8 running locally
- Maven 3.9+
